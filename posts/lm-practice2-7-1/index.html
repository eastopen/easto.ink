Lm Practice2: 7-1 Calculate factorial sum

Mar 26, 2024

Question

For a given positive integer N, you need to calculate S=1!+2!+3!+…+N!.

Input

The input gives a positive integer N not exceeding 10 in a row.

Output

Print the value of S in one line.

Sample Input

3

Sample Output

9

Answer (C)

Loop to find factorial.

#include <stdio.h>

int get(int num)
{
    int sum = 1;
    for (int i = 1; i <= num; i++)
        sum *= i;

    return sum;
}

int main()
{
    int i = 0;
    scanf("%d", &i);

    int sum = 0;
    for (int j = 1; j <= i; j++)
        sum += get(j);

    printf("%d\n", sum);
}
