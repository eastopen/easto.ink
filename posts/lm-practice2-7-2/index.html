Lm Practice2: 7-2 So handsome that he has no friends

Mar 26, 2024

Question

When all living beings are busy posting photos in WeChat Moments, there are always some people who have no friends because they are too handsome. This question asks you to find those people who are so handsome that they have no friends.

Input

The first line of input gives a positive integer N (≤100), which is the number of known friend circles; the following N lines, each line first gives a positive integer K (≤1000), which is the number of people in the friend circle, and then List all the people in a circle of friends - for convenience, each person corresponds to an ID number, which is a 5-digit number (from 00000 to 99999), and the IDs are separated by spaces; then a positive integer M (≤10000) is given , is the number of people to be queried; the following row lists M IDs to be queried, separated by spaces.

Note: People who have no friends can either have no “Moments” installed at all, or they can be the only ones in the Moments. Although some narcissists will add themselves to their friend circles repeatedly, the question ensures that all friend circles with K exceeding 1 will have at least 2 different people in them.

Output

Output those guys who are so handsome that they have no friends, in the order they were entered. Separate IDs with 1 space, and there must be no extra spaces at the beginning or end of the line. If no one is handsome, output No one is handsome.

Note: The same person can be queried multiple times, but will only be output once.

Sample Input

3
3 11111 22222 55555
2 33333 44444
4 55555 66666 99999 77777
8
55555 44444 10000 88888 22222 11111 23333 88888

Sample Output

3
3 11111 22222 55555
2 33333 44444
4 55555 66666 99999 77777
4
55555 44444 22222 11111

Answer (Python)

There are these issues that need attention

  • Maybe only 1 item in the “Moments”.
  • Maybe have multiple same items in the “Moments”.
  • Output in order of input, but cannot be repeated.
count = int(input())
m = {}

for _ in range(count):
    l = input().split(" ")
    if l[0] != "1" and len(set(l[1:])) != 1:
        for j in set(l[1:]):
            m[j] = True

input()

cm = {}
f = False

for i in input().split(" "):
    if not m.get(i, False) and not cm.get(i, False):
        cm[i] = True
        print((" " if f else "") + i, end="")
        f = True

if not f:
    print("No one is handsome")
